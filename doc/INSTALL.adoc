= Installing GruntWurk Core for Python

== Stable Version *(recommended)*

(NOT YET)

[source,bash]
----
python38 -m pip install gwpycore
----

== Bleeding-Edge Version

[source,bash]
----
python38 -m pip install https://github.com/gruntwurk/gwpycore/archive/develop.zip
----
